\doxysubsection{Detailed Description}
Класс \mbox{\hyperlink{classSettingsDialog}{Settings\+Dialog}} представляет диалоговое окно для настройки параметров. 

Класс позволяет пользователю вводить значения температуры, влажности и давления, а также проверяет корректность введенных данных. При подтверждении данных испускается сигнал с новыми значениями параметров.
