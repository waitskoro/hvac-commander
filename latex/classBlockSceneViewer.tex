\doxysubsection{Detailed Description}
Класс \mbox{\hyperlink{classBlockSceneViewer}{Block\+Scene\+Viewer}} представляет графическую сцену для управления блоками и отображения данных. 

Класс наследуется от QGraphics\+Scene и предоставляет функциональность для управления блоками, отображения температуры и взаимодействия с пользователем через кнопки.
