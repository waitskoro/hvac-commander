Detailed Description

Класс BlockSceneViewer представляет графическую сцену для управления блоками и отображения данных.

Класс наследуется от QGraphicsScene и предоставляет функциональность для управления блоками, отображения температуры и взаимодействия с пользователем через кнопки.
